# SnowflakeBasicETL
Snowflake basic ETL project

Project Overview

The project loads raw json data into Snowflake, applies data transformations to create a clean and structured dataset, and produces analytics-ready tables for reporting.

Key Objectives:
1. Set up Snowflake schemas and warehouses
2. Load raw data into a staging layer
3. Transform data into analytics tables
4. Run queries to generate insights

Small python script was used to generate json files with random blood test data. Generated files were uploaded to stage.
